O Handbook é um guia prático para times que querem usar AI coding assistants em projetos reais e em escala — não como autocomplete sofisticado, mas como parte do processo de desenvolvimento.

The Handbook is a practical guide for teams that want to use AI coding assistants in real projects at scale — not as sophisticated autocomplete, but as part of the development process.

É denso por design: cada seção é independente. Leia linearmente uma vez para o mapa mental; depois consulte por seção quando aparecer o problema específico.

It is dense by design: each section is self-contained. Read it linearly once for the mental map; then consult section by section when the specific problem appears.
